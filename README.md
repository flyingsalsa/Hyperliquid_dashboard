# HL Dashboard

A simple dashboard with tabs built with HTML, CSS, and JavaScript, served via Node.js.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Development

The server will automatically serve all static files (HTML, CSS, JS) from the project directory.
